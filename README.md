# Vincent Pickering [blog]

## Setup

To install all global packages. There are no local packages.

```
./install.sh
```

## Running

Launch the pleeease watch in one window from the root.

```
pleeease watch
```

And in a separate window run jekyll

```
jekyll serve --incremental --drafts
```

## Deploy

Package Jekyll

```
jekyll build
```

## Testing Tools

Run tests

```
./run-tests.sh
```
